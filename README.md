# 🎮 Maritime

**Maritime** est un jeu vidéo réalisé dans le cadre d’un projet académique par Jack-Arthur Zue, Kenan Ammad et Gauthier Defrance, étudiants en deuxième année de licence MIPI à l’Université de Cergy-Pontoise, sous la direction de M. Tianxiao Liu.

## 🌊 Objectif du jeu

S'emparer de tout les ports présents sur la carte en devenant extrêmement riche.

## 🛠️ Technologies utilisées

- **Langage de programmation** : Java
- **Graphismes** : Pixilart et Paint
- **Gestion de version** : Git + GitHub

## 🎨 Principales fonctionnalités

- Grande quantité de menu
- Système d'économie et de route commerciales
- Système de bataille navale
- Système de diplomatie
- Musique immersive et variant selon les situations
- et plus encore.

## 📦 Utilisation

Aucune version .jar n'est disponible malheureusement.
Il vous faudra vous équiper d'une version java 21 et lancer votre MainGui.java dans votre environnement de programmation tel que IntelliJ.
